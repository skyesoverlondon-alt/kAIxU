How your apps should call the gate

fetch("https://YOUR-GATE.netlify.app/v1/generate", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_APP_TOKEN"
  },
  body: JSON.stringify({
    model: "kAIxU6.7-flash",
    input: { type: "text", content: "Hello" },
    generationConfig: { temperature: 0.7 }
  })
})

Notes

This page is only a console + status. Your real “AI home” is the gateway API. Use Git deploy so the functions run reliably.

Hardening checklist

  • Keep KAIXU_OPEN_GATE = 0
  • Use unique app tokens per product
  • Set KAIXU_ALLOWED_ORIGINS to your domains
  • Rotate tokens quarterly
